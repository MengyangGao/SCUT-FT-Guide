# 实验四《时序逻辑电路设计》预习报告

## 一、实验目的

1. 掌握使用 J-K 触发器设计同步计数器的方法；
2. 熟悉交通灯时序控制电路的设计流程；
3. 了解在 Quartus II 上用图形输入法与文本输入法实现数字电路的基本方法；
4. 学会 FPGA 开发板的引脚配置与程序烧录流程。

## 二、实验器材

- 计算机一台；
- Quartus II 13.1 开发环境；
- Cyclone IV FPGA 开发板；
- 若干杜邦线；
- 实验箱及 LED 接口。

## 三、实验内容概述

设计并实现一个基于三位二进制同步计数器的交通灯控制系统，南北方向与东西方向交替控制红、黄、绿灯的亮灭，具体时序为：红灯亮3秒 → 绿灯亮3秒 → 黄灯亮1秒。共需使用3个J-K触发器和6个与非门，构成计数器和控制电路。

## 四、时序控制逻辑功能分析

### 4.1 状态分配

使用三位二进制同步计数器 Q3 Q2 Q1 编码共8个状态（000 至 111），每个状态对应不同交通灯组合：

| 状态 | Q3Q2Q1 | 南北灯状态 | 东西灯状态 | 说明     |
| ---- | ------ | ---------- | ---------- | -------- |
| S0   | 000    | 绿（C1）   | 红（A2）   | 南北通行 |
| S1   | 001    | 绿（C1）   | 红（A2）   | 南北通行 |
| S2   | 010    | 绿（C1）   | 红（A2）   | 南北通行 |
| S3   | 011    | 黄（B1）   | 红（A2）   | 南北过渡 |
| S4   | 100    | 红（A1）   | 绿（C2）   | 东西通行 |
| S5   | 101    | 红（A1）   | 绿（C2）   | 东西通行 |
| S6   | 110    | 红（A1）   | 绿（C2）   | 东西通行 |
| S7   | 111    | 红（A1）   | 黄（B2）   | 东西过渡 |

### 4.2 状态转移图

状态按照上述顺序循环：S0→S1→S2→S3→S4→S5→S6→S7→S0。

### 4.3 计数器设计思路

- 采用三个 J-K 触发器构成同步计数器；
- 编写真值表确定每个触发器输入 J、K 的驱动方程；
- 使用与非门组合实现这些驱动逻辑函数；
- 用计数器输出 Q1、Q2、Q3 通过组合逻辑控制六盏交通灯的点亮状态。

### 4.4 触发器驱动方程（预设）

将三位状态编号 Q1Q2Q3 代入状态转移表，推导出如下激励表，化简得出：

- J1 = K1 = 1 （Q1 每次取反）  
- J2 = K2 = Q1  
- J3 = K3 = Q1·Q2

## 五、预期实验步骤简述

1. 在 Quartus II 中使用图形设计法构建上述计数器与交通灯控制电路；
2. 编译并查看 RTL 图，确保逻辑结构符合设计预期；
3. 进行功能仿真，观察 Q1~Q3 与灯状态随时钟变化的波形；
4. 在 FPGA 开发板上进行引脚分配与烧录，观察实物灯光交替变化；
5. 对照逻辑分析仪或 LED 显示确认实验逻辑与仿真一致。

## 七、预期问题思考

1. 若将红绿灯持续时间扩展为 30 秒（原为3秒），可通过降低系统输入时钟频率或使用更高位数的计数器进行分频实现。
2. 图形输入法直观易学，适合初学者；文本输入法更高效、便于移植与批量修改，适合大项目。
